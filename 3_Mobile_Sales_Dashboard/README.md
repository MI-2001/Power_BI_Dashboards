## Overview

This Power BI dashboard presents a complete view of **mobile phone sales performance** across brands, models, cities, payment methods, and time.  
The goal of this project is to turn raw transactional data into **clear, business-ready insights** that help understand sales trends, customer behavior, and product performance.

## Dataset Overview

The dataset is based on mobile sales transactions and is structured to support analytical reporting and interactive exploration.

**Key columns include:**
- Date and Day Name  
- Brand and Mobile Model  
- City  
- Payment Method  
- Customer Age and Customer Ratings  
- Total Sales, Total Quantity, Transactions, Units Sold, Price per Unit  

The data model allows analysis from **high-level KPIs down to detailed product and city-level insights**.
